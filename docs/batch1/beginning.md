---

Tujuan Batch

Batch pertama adalah fondasi dari seluruh proyek.

AI wajib memahami bahwa proyek ini bukan sekadar dashboard monitoring VPS.

Proyek ini adalah platform modern untuk mengelola VPS secara penuh melalui antarmuka web yang cepat, stabil, responsif, aman, modern, mobile-first, realtime, modular, dan mudah dikembangkan.

Seluruh keputusan yang dibuat pada batch ini akan menjadi dasar seluruh batch berikutnya.

AI tidak boleh mengubah keputusan arsitektur tanpa instruksi baru.

---

Visi Produk

My Dash adalah dashboard VPS modern yang menggabungkan monitoring realtime, manajemen server, tunnel otomatis, analitik, notifikasi pintar, keamanan, backup, automation, serta asisten AI dalam satu aplikasi.

Alur Instalasi

Pengguna menjalankan installer menggunakan bash.

Installer melakukan pemeriksaan sistem.

Installer menentukan dependency.

Installer meminta password dashboard.

Installer menghasilkan recovery key.

Installer meminta informasi owner.

Installer meminta tiga persetujuan.

Installer menginstal seluruh komponen.

Installer membuat tunnel otomatis.

Installer menampilkan URL dashboard.

---

Password Dashboard

Dashboard tidak menggunakan username.

Dashboard hanya menggunakan password utama.

Password dibuat ketika instalasi pertama.

Password harus di-hash.

Password asli tidak boleh disimpan.

Session default berlaku 24 jam.

Setelah habis wajib login kembali.

---

Recovery

Installer membuat Recovery Key.

Recovery Key hanya ditampilkan satu kali.

Recovery Key digunakan apabila password lupa.

Apabila Recovery Key hilang maka reset hanya dapat dilakukan melalui terminal dengan hak akses root.
